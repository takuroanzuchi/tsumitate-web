infoアプリの概要

このアプリは、あなたの投資計画をサポートする2つの主要機能を提供します：

savings積立目標管理

具体的な目標を設定し、積立計画の進捗を追跡します。実際の資産額を入力して達成率を確認できます。

pie_chartポートフォリオシミュレーション

投資期間や月々の積立額を設定して、将来の資産価値をシミュレーションできます。

savings積立目標の使い方

add_circle新しい目標を作成する

  1. ホーム画面の「目標を追加」ボタンをタップします
  2. 目標名を入力します（例：「マイホーム購入資金」）
  3. 目標金額を設定します
  4. 毎月の積立予定額を入力します
  5. 目標達成予定日を設定します
  6. 「保存」ボタンで目標を作成完了です

tips_and_updatesポイント

現実的な目標設定が重要です。無理のない月々の積立額から始めましょう。

trending_up進捗を確認・更新する

  1. 目標一覧から確認したい目標をタップします
  2. 詳細画面で現在の達成率を確認できます
  3. 「実際の金額を更新」から正確な資産額を入力します
  4. グラフで進捗の推移を視覚的に確認できます

show_chart進捗の見方

  • 青色（0-49%）：まだ目標達成まで時間がかかります
  • オレンジ色（50-79%）：順調に進捗しています
  • 緑色（80%以上）：目標達成まであと少しです

pie_chartポートフォリオシミュレーション

calculateシミュレーションを作成する

  1. シミュレーション画面の「新規作成」ボタンをタップします
  2. シミュレーション名を入力します
  3. 投資期間（年数）を設定します
  4. 毎月の積立予定額を入力します
  5. 期待リターン（年率）を設定します
  6. 「シミュレーション実行」で結果を表示します

warning注意事項

シミュレーション結果は予想値です。実際の投資結果は市場状況により異なります。

insights結果の見方

  • 総投資額：期間中に積立てる合計金額
  • 予想評価額：期間終了時の予想資産価値
  • 利益率：投資による利益の割合
  • グラフ：時間経過による資産成長の推移

lightbulb効果的な使い方のコツ

schedule定期的な見直し

月に1回は目標の進捗を確認し、必要に応じて計画を調整しましょう。

balance複数目標の管理

短期・中期・長期の目標を分けて管理することで、バランスよく資産形成できます。

compareシミュレーション比較

異なる条件でシミュレーションを作成し、最適な投資戦略を見つけましょう。

backupデータの管理

重要な目標やシミュレーション結果は定期的にバックアップを取りましょう。

psychology投資の心構え

  • 長期的な視点を持つことが重要です
  • 市場の短期的な変動に一喜一憂しないようにしましょう
  • 定期的な積立（ドルコスト平均法）でリスクを分散させます
  • 投資は余裕資金で行いましょう

helpよくある質問

help_outline目標を削除したい場合は？

目標詳細画面から「編集」→「削除」で目標を削除できます。削除した目標は復元できませんので注意してください。

help_outlineシミュレーション結果が保存されない

シミュレーション作成時に名前を入力し、「保存」ボタンを押すことで結果が保存されます。

help_outlineデータのバックアップは？

現在、データは端末に保存されます。機種変更時はデータ移行機能をご利用ください。
